Hi! For my University coursework I have created the soundscape for the open source game Viking Village. In addition to creating, processing and implementing sounds into the original game I also made some changes to the game level itself. I have set up a treasure hunt style mini game which guides the player through the different locations of the game to make the experience more interesting. I used Wwise and Unity to process and implement sounds in conjunction with Visual Studio to edit the necessary scripts. I have created all audio assets with the Digital Workstations Pro Tools and Reaper.

I aimed to expand the setting of a viking village to make it sonically more interested and therefore added several event location like for example a pirate ship, zombie temple, dragon tower and jungle. I have created the final assets from a mixture of sounds found on freesound.org and commercially released sound effects libraries and edited them together with my own recordings.

This video shows the additional assets I have added in Unity and explains how I approached the implementation of sounds in Unity and Wwise. Moreover, I explain how I used C# scripting, game parameters, sound events and SoundBanks to trigger the right sound at the right moment.
